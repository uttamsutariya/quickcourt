### QuickCourt

QuickCourt is a platform that enables sports enthusiasts to book local sports facilities (e.g., badminton courts, turf grounds, tennis tables) and create or join matches with others in their area.

## Technologies

### Backend

- Node.js
- Express.js
- MongoDB
- Mongoose
- WorkOs for auth
- Typescript

### Frontend

- React.js
- Zustand
- React-Router-Dom
- Tailwind & ShadCN UI
- WorkOs for auth
- Typescript
